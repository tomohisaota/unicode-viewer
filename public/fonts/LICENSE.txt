CJK IVS Fonts
=============

This directory contains two derived font families for displaying CJK
Ideographic Variation Sequences (IVS) from different collections.


1. CjkIvsMincho (Moji_Joho collection)
---------------------------------------
Derived from IPAmj明朝 (IPAmj Mincho) version 006.01.
Copyright (c) Information-technology Promotion Agency, Japan (IPA)
Licensed under the IPA Font License Agreement v1.0.
https://opensource.org/licenses/IPA

Original: https://moji.or.jp/mojikiban/font/


2. CjkAj1Mincho (Adobe-Japan1 collection)
------------------------------------------
Derived from Source Han Serif JP (源ノ明朝) version 2.003.
Copyright 2017-2024 Adobe (http://www.adobe.com/)
Licensed under the SIL Open Font License, Version 1.1.
https://openfontlicense.org

Original: https://github.com/adobe-fonts/source-han-serif


Modifications
-------------
Both fonts are subset into WOFF2 chunks by Unicode range (256 code
points each) for on-demand web loading via CSS unicode-range.
Font family names are changed per IPA Font License Article 3(4).
