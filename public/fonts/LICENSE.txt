CJK IVS Fonts
=============

The `cjk-ivs-*.woff2` files in this directory form a single derived web
font used by the Unicode Viewer site to render CJK Ideographic Variation
Sequences (IVS) and Standardized Variation Sequences (SVS). The font is
assembled from four upstream sources and subset into WOFF2 chunks by
Unicode range (256 code points each) for on-demand loading via CSS
`unicode-range`.

The merged font is distributed under the most restrictive of its inputs
(IPA Font License v1.0, for the SVS glyphs carried over from IPAmj).


1. Jigmo — base glyphs and all IVS
------------------------------------
Version 2025-09-12 (implements IVD 2025-07-14).
Copyright (c) Koichi Kamichi and GlyphWiki contributors.
Licensed under CC0 1.0 Universal (Public Domain Dedication).
https://creativecommons.org/publicdomain/zero/1.0/

Provides the base cmap and every Ideographic Variation Sequence
(U+E0100-U+E01EF) registered in the Unicode IVD as of 2025-07-14.
Glyphs are generated from GlyphWiki via the KAGE system.

Original: https://kamichikoichi.github.io/jigmo/
GlyphWiki: https://glyphwiki.org/


2. Source Han Serif JP (源ノ明朝) — SVS overlay (primary)
----------------------------------------------------------
Version 2.003.
Copyright 2017-2024 Adobe (http://www.adobe.com/).
Licensed under the SIL Open Font License, Version 1.1.
https://openfontlicense.org

Provides the Adobe-Japan1 SVS entries that Jigmo lacks — most notably
the U+FE01 "centred" variants of CJK punctuation (e.g. 、︁ U+3001+FE01,
。︁ U+3002+FE01) used in vertical writing.

Original: https://github.com/adobe-fonts/source-han-serif


3. IPAmj明朝 (IPAmj Mincho) — SVS overlay (Moji_Joho-specific)
---------------------------------------------------------------
Version 006.01.
Copyright (c) Information-technology Promotion Agency, Japan (IPA).
Licensed under the IPA Font License Agreement v1.0.
https://opensource.org/licenses/IPA

Layered after Source Han Serif to fill any Moji_Joho-specific SVS
entries SHS doesn't cover.

Original: https://moji.or.jp/mojikiban/font/


4. Noto Sans Math — Math operator base + SVS
-----------------------------------------------
Copyright (c) Google / Noto Project.
Licensed under the SIL Open Font License, Version 1.1.
https://openfontlicense.org

Supplies math operator code points (∩ U+2229, ∪ U+222A, ∅ U+2205, etc.)
and their SVS variants ("with serifs" forms at U+FE00) — Jigmo, IPAmj,
and Source Han Serif do not cover the U+22xx mathematical operator
range.

Original: https://fonts.google.com/noto/specimen/Noto+Sans+Math


Merge priority
--------------
Base cmap and all IVS (U+E0100-U+E01EF) are taken from Jigmo so that
every glyph in the final font is drawn by the same typeface — making
base-vs-variant overlays in the Unicode Viewer UI show structural
differences without typographic style drift. IPAmj's SVS entries are
layered on top only because Jigmo ships no SVS table.


Modifications
-------------
* UPM is normalised to Jigmo's 1024 during IPAmj SVS glyph copy using
  TransformPen; glyph metrics (hmtx, vmtx) are scaled accordingly.
* The merged font is subset in-memory to the Unicode ranges this site
  ships (Basic Latin, math operators, misc symbols, CJK blocks, CJK
  compat, halfwidth/fullwidth forms, plus VS ranges).
* WOFF2 chunks are produced with `fonttools subset --layout-features='*'`.
* Family name is changed from the originals per IPA Font License
  Article 3(4).
